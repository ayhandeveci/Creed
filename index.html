Formül referansı

Sınırlı dalgalanma — tam kredibilite

λ_F = ( zp / r )²  beklenen hasar

zp, p güveni için iki taraflı normal değerdir; r tolerans. Poisson hasar sayısı varsayar (Var = ortalama). Toplam/saf prim için (1 + CV²şiddet) ile çarp.

Kısmi kredibilite (karekök kuralı)

Z = min( 1 , √( n / λ_F ) )

n = fiilen gözlemlenen hasar sayısı. Standardın altındayken verin yalnızca kısmi kredi alır.

Kredibilite tahmini

Tahmin = Z·x̄ + (1−Z)·μ

Gözlemlenen ortalamayı manuel/önsel ile harmanla. Hem klasik hem Bühlmann için aynı yapı.

Bühlmann (en yüksek doğruluk)

k = EPV / VHM = E[Var(X|θ)] / Var(E[X|θ])
Z = n / (n + k)

EPV = beklenen süreç varyansı (bir risk içindeki gürültü). VHM = hipotetik ortalamaların varyansı (riskler arası yayılım). n→∞ iken Z→1. Bühlmann-Straub: n yerine toplam maruziyet m kullan.

EPV & VHM hesaplama

μ(θ)=E[X|θ],   v(θ)=Var(X|θ)
EPV = E[ v(θ) ] = Σ P(θ)·v(θ)
VHM = Var( μ(θ) ) = E[μ²] − (E[μ])²

Poisson kısayolu: v(λ)=μ(λ)=λ, yani EPV = E[Λ] ve VHM = Var(Λ). Veriden: EPV = risk içi örneklem varyanslarının ortalaması; VHM = (risk ortalamalarının örneklem varyansı) − EPV/n.
